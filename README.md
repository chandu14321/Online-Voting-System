# 🗳️ VoteFlow — Modern MERN Online Voting Platform

A modern full-stack voting platform built using the MERN stack with secure authentication, Redux state management, interactive poll visualization, and a redesigned Apple-inspired minimalist UI.

Designed to simulate a scalable real-world voting application with secure backend architecture and a premium frontend experience.


# 🔧 Backend Setup

## Install Dependencies

```bash id="qbjk48"
cd server
npm install
```

## Create `.env` File

Inside the `server` folder:

```env id="y4u7bc"
PORT=5000
MONGO_URI=your_mongodb_connection
SECRET_KEY=your_secret_key
```

## Start Backend Server

```bash id="6t9iq2"
npm run dev
```

# 📊 API Overview

| Method | Endpoint             | Description   |
| ------ | -------------------- | ------------- |
| POST   | `/api/auth/register` | Register User |
| POST   | `/api/auth/login`    | Login User    |
| GET    | `/api/polls`         | Fetch Polls   |
| POST   | `/api/polls`         | Create Poll   |
| POST   | `/api/vote`          | Cast Vote     |
